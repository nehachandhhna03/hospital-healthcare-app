# 🏥 Hospital Healthcare App

## 📌 Project Overview

The Hospital Healthcare App is a simple desktop application developed using Python and the Tkinter library. It provides users with an easy-to-use interface to access essential healthcare services such as appointment booking, medicine reminders, and health records.

## 🎯 Objective

The objective of this project is to simplify basic hospital-related services through a user-friendly graphical interface.

## ✨ Features

- 📅 Book Doctor Appointment
- 💊 Medicine Reminder
- 📋 View Health Records
- ℹ️ About the Application
- ❌ Exit Application

## 🛠️ Technologies Used

- Python 3
- Tkinter (GUI)

## 📂 Project Structure

```
Hospital-Healthcare-App/
│── hospital_app.py
│── README.md
│── requirements.txt
```

## ▶️ How to Run

1. Install Python 3.
2. Open the project folder.
3. Run the application:

```bash
python hospital_app.py
```

## 📸 Output

The application provides a graphical user interface where users can:
- Book appointments
- View health records
- Receive medicine reminders
- Learn about the application
